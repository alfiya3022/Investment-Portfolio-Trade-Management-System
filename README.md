# Investment Portfolio Analytics & Trade Data Warehouse

## Overview

Investment Portfolio Analytics & Trade Data Warehouse is an end-to-end SQL Server and Power BI project that simulates a real-world investment management platform. The solution integrates investor portfolios, stock trades, broker performance, dividend analysis, and historical market data into a centralized analytical data warehouse for business intelligence and decision-making.
